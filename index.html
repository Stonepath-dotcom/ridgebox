Keamanan

Data kamu, kendali kamu

Arsitektur direct-upload memastikan file tidak melewati server perantara manapun.

Upload Langsung

File dikirim langsung dari browser ke Telegram API. Tidak ada server perantara yang menyimpan atau memindai file kamu. Bandwidth Vercel tidak terpakai untuk upload.

AES-256-GCM Encryption

Enkripsi end-to-end menggunakan Web Crypto API dengan PBKDF2 key derivation 100K iterasi. Password tidak pernah dikirim ke server mana pun.

Share Link Password

Lindungi link download dengan password. Penerima harus memasukkan password yang benar sebelum bisa mengunduh file.

Self-Destruct Link

Link yang otomatis expired setelah 1x download atau dalam waktu tertentu. Tidak ada jejak setelah kadaluarsa.

FAQ

Pertanyaan Umum

Apakah Ridgebox benar-benar gratis?
Ya, 100% gratis. Ridgebox menggunakan Telegram Bot API sebagai backend storage. Kamu hanya perlu Telegram Bot (gratis dari @BotFather) dan akun Telegram. Tidak ada biaya server, tidak ada subscription, dan tidak ada iklan. Semua 45 fitur bisa digunakan tanpa batas.
Berapa batas ukuran file yang bisa diupload?
Telegram Bot API mendukung upload hingga 50 MB per file secara default, dan bisa sampai 2 GB menggunakan metode yang sesuai. Ridgebox mendukung keduanya — kamu bisa atur batas upload di pengaturan dashboard. File besar akan di-upload langsung dari browser ke Telegram API tanpa melewati server perantara.
Apakah file saya aman?
Keamanan file bergantung pada lapisan yang kamu gunakan. Secara default, file disimpan di server Telegram yang sudah terenkripsi. Jika kamu aktifkan enkripsi Ridgebox, file akan dienkripsi dengan AES-256-GCM sebelum dikirim, sehingga bahkan Telegram tidak bisa membaca isinya. Password enkripsi tidak pernah dikirim ke server manapun.
Apa yang terjadi jika Telegram down?
Ridgebox menggunakan IndexedDB di browser untuk menyimpan metadata file secara lokal. Kamu tetap bisa melihat daftar file dan catatan meskipun offline. Namun, upload dan download membutuhkan koneksi ke Telegram API. Begitu Telegram kembali online, semua fitur berfungsi normal kembali.
Bisa pakai lebih dari satu bot?
Ya! Ridgebox mendukung multi-bot. Kamu bisa mengkonfigurasi beberapa Telegram Bot dan berpindah antar bot langsung dari dashboard. Ini memungkinkan kamu memisahkan storage personal dan storage kerja, misalnya, atau menggunakan bot berbeda untuk kategori file yang berbeda.
Bagaimana cara setup Telegram Bot?
Buka Telegram, cari @BotFather, kirim /newbot dan ikuti instruksinya. Setelah bot dibuat, kamu akan mendapatkan Bot Token. Lalu, buat group atau channel, tambahkan bot, dan forward pesan apapun ke bot untuk mendapatkan Chat ID. Masukkan kedua nilai ini ke environment variable Vercel (TG_BOT_TOKEN dan TG_CHAT_ID). Selesai!
Apakah bisa di-install di HP?
Ya, Ridgebox adalah Progressive Web App (PWA). Buka di browser mobile, dan akan muncul opsi "Install ke Homescreen". Setelah di-install, Ridgebox berfungsi seperti app native dengan ikon sendiri, tanpa address bar, dan bisa dibuka langsung dari homescreen.
